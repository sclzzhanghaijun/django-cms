PD-CMS是一款基于Django框架开发的内容管理系统(CMS)。其主要特点是简单易用，性能高，便于进行二次开发。

PD-CMS特性和说明：
    1、后台界面主要使用layuiAdmin框架进行开发，界面美观大气；
    2、强大的文档管理；
    3、简单、易配置的权限管理；
    4、架构简单明了，逻辑清楚，友好的代码风格；
    5、核心思想：化繁为简，简单易用
    6、只做基础框架，便于后期的二次的开发；
